本地博客搭建

安装git

下载 msysgit 并执行即可完成安装。

安装Node.js

在 Windows 环境下安装 Node.js 非常简单，仅须下载安装文件并执行即可完成安装。

安装hexo

一个台湾人写的markdown静态博客框架，使用NodeJS实现，速度快，轻量级。
利用 npm 命令即可安装。（在任意位置点击鼠标右键，选择Git bash）

错误：hexo:command not found
删除刚刚安装的npm目录，重新执行命令npm install -g hexo安装hexo

创建hexo文件夹根目录

D:\Hexo 右键，选择Git bash

安装依赖包

本地预览

浏览器输入localhost:4000
至此，本地博客已经搭建起来

部署到github

创建github博客仓库

这个仓库的名字需要和你的账号对应，格式: yourname.github.io

生成sshkey秘钥

cmd命令输入 ssh-keygen -t rsa -b 4096 -C “your email“
在C:\Users.ssh下，得到两个文件id_rsa和id_rsa.pub

在GitHub上添加SSH密钥

打开id_rsa.pub，复制全文。https://github.com/settings/ssh ，Add SSH key，粘贴进去。

测试一下 ssh -T git@github.com
Hi w5pand! You’ve successfully authenticated ……表示成功

hexo目录说明

├── .deploy #需要部署的文件
├── node_modules #Hexo插件
├── public #生成的静态网页文件
├── scaffolds #模板
├── source #博客正文和其他源文件，404、favicon、CNAME 都应该放在这里
| ├── _drafts #草稿
| └── _posts #文章
├── themes #主题
├── _config.yml #全局配置文件
└── package.json

hexo全局配置 _config.yml

**注意：配置文件的冒号“:”后面有一个空格
deploy举例：

deploy:
type: git
repository: git@github.com:w5pand/w5pand.github.io.git
branch: master
